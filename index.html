Useful tools:

  • Loebolus

    Making the public domain Loebs more easily downloadable.

  • Latin OCR for Tesseract

    Latin OCR training data and tools for Tesseract.

  • Pleiades Static

    Fast, static, client-side search and display for the Pleiades gazetteer of ancient places.

  • GitHub Friction

    Check for common sources of contributor friction across your GitHub repositories.

  • Beer Calories

    Client-side mobile-friendly beer calorie calculator with Untappd integration.

  • dlflickr

    Ruby scripts for downloading Flickr photos and associated metadata.

  • iiif-dl

    Ruby script for downloading IIIF images.

  • dzi-dl

    Ruby script for downloading Deep Zoom images.

  • google-fusion-tables-backup

    Ruby script for backing up your Google Fusion Tables data to CSV.

  • GeoRefine

    A geospatial processing extension for OpenRefine.

  • RotateBake

    RotateBake is a Mac app for baking EXIF rotation into images. You know your iPhone images that show up rotated wrong in Safari? Yeah, this fixes that.

  • Macduff

    Macduff is a tool for finding the Macbeth ColorChecker chart in an image.

  • gist2pinboard

    Script for adding your GitHub Gists as Pinboard bookmarks, so that you can search your own Gists.

  • Backpack Export XSLT

    Stylesheets for converting your 37signals/Basecamp Backpack export XML into usable, complete, static local backups.

  • BCX Project Photocopier

    Copy Basecamp projects between accounts.

  • Manuscript Images on Flickr

    Creative Commons licensed manuscript images mirrored on Flickr for redundancy and alternative access methods (API, notes, comments, browsing, etc.).
