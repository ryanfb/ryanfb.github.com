    Ruby script for downloading IIIF images.

    Ruby script for downloading Deep Zoom images.

  • bl-dl

    Script for downloading manuscript images from the British Library.

  • dlflickr

  • google-fusion-tables-backup

    Macduff is a tool for finding the Macbeth ColorChecker chart in an image.

  • Manuscript Images on Flickr

    Creative Commons licensed manuscript images mirrored on Flickr for redundancy and alternative access methods (API, notes, comments, browsing, etc.).
